# Webデザイナーの仕事探し
パソコンショップのアルバイトをしながらWebデザインの勉強をやり始めて1年ほどたったあたりから、仕事を探し始めた。

年齢的にはちょうど新卒で就職する時期である。ただし実務経験のない自分が正社員での求人に応募するのはなかなかハードルが高かった。
数社のWebデザイナーの求人に応募してみたもののはやり不採用。

そんななか偶然ある自宅の近くの社員数5名ほどの小さなWeb制作会社でアルバイトを募集していた。実務経験を得るにはちょうど良い求人である。応募して面接を受けてみたところ運よく合格。

苦戦することを覚悟していたが、アルバイトとはいえ想定よりも早く仕事にありつけた。入社後半年くらいで社員にもなることができた。大学の同級生たちよりも少し早いタイミングでの就職となった。就職難であったこの時期にどうなるかと思ったが、御の字である。

# プロの洗礼を受ける
入社初日、社長からいろいろ質問を受ける。

```
社長：これとこれとこれとこれとこれとこれとこれとこれとこれとこれとこれ...できる？
著者： これはできます。これはできません。これもできません。これはできます。これは...
社長： いままで何勉強してきたの？
著者： う...
```

仕事で使うツール類についての質問だったのだが、圧倒的に知識不足であることを思い知らせれる。はやり**独学+スクールの学習ではプロの現場では圧倒的にスキル不足**だった。

仕事内容は、地元密着で中小企業のホームページの制作代行や更新作業。仕事は大変だったものののひとつづつスキルを身に付けて着実に実務経験を積むことはできた。

# 職場環境はばっちりだけど...
会社は社長の自宅兼事務所でSOHOスタイルの仕事環境。SOHOとは「Small Office Home Office」の略語で、自宅兼事務所で少人数で働くスタイル。個人事業主や小規模事業者の間で当時はやり始めていた、自分もあこがれていたワークスタイルです。

職場環境は良かったがとにかく激務で大変だった。会社がブラックというよりも客からの無茶ぶりが多かった印象だ。

なにしろまだ世間的にもインターネットやホームページというものが理解されていない時代だ。ホームページのイメージカラーを「赤」から「緑」に変更がどれだけ大変かなんて説明しても理解してもらえない。 **「プロなんだから色の変更くらい5分でタダでやれよ」** なんてことを客が平気で言ってくる。

それどころか、**故障した客のパソコンのことまでWeb制作会社であるうちの会社に文句言ってくる始末** である。買った店に聞けよ。

客とこんなやり取りを毎日やりながらの制作業務がスムーズに進むわけがない。そんな感じで客からの無茶ぶりに翻弄されながらも着実に実務経験は積んでいった。

# あっさり廃業
「Web制作業は廃業してビジネスコンサルに業態を変更するよ。これからは制作じゃなくて営業やってね」

入社して1年ほどたったある日、社長がこう言い出した。
会社がつぶれたということではなく、社長がもっと儲かりそうな仕事をしたいということだ。

大学を中退してまでも避けて通った営業職だ。やっとつかんだWebデザイナーの実務経験は無駄にはできない。「実務経験1年」の経歴を堂々と履歴書に書けるので、どうにかなるだろうと小さな自信とともに退職届を提出した。

こうしてあっさり1社目の会社を退職することになる。**この時点ではIT業界は文系でも大丈夫じゃん～って思っていたが、のちに痛い目を見る。**
