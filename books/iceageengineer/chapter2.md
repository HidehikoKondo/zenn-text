# 高校時代の家庭環境
SHARP x68000が自宅にあった。PCゲームで遊んだ。ファミコンを買ってもらえない家庭だったため、ゲーム機の代替として楽しんだ。このおかげですっかりパソコン少年になった。

# 高校生活
平凡なレベルの県立高校に入学。いわゆる「管理教育」と呼ばれる悪しき教育方針の名残のある居心地の悪い学校だった。

コンピュータ部に入部。部室にあったNECのPC88で、BASICでゲームらしきものを作った。ここではじめてプログラミングを体験した。ここで何となくIT業界で将来仕事をするかもしれないとうっすら感じ始めたかもしれない。

数学が特に苦手で赤点常連。文系コースへ進む。英語と歴史の授業は好きだった。

# 進路指導
「IT系の専門学校に行きたい」というのが自分の希望した進路。興味のない大学受験するよりも自分の興味のある専門分野で技術を身に着けたいと考えたからだ。

それに対すて先生と親は強く言ってきた。
「大学に行けるだけの成績があるのだから大学に行け。専門学校が大学に行けないやつが行くところだ。」

当時の自分には親や先生を説得するほどのITスキルに自信もなく、仕方なく従った。

ここで、ささやかな反抗としてパソコンショップでアルバイトをこっそり始める。
そもそも進学する気のない大学への受験勉強に身が入るわけもなく、卒業後だれでも合格できるFランク大学の商業学部へ進学することになる。

# 大学生活
なんとなく通い続けた大学の2年生になったころ、就職活動に関する指導も学校から始まった。
しかし時は就職難真っただ中で、新卒での就職率は歴代最低水準。
先輩方は何十社も履歴書を送ってもほとんど面接にすら進めない状況。
商業学部ということもあり就職先はほぼすべて営業職。自分の望む職業ではない。
この大学を卒業したところで、就職できるかどうかすらわからず、就職できたところで望んだ職業には就けないことが分かり、これが決め手で中退をすることになった。

# パソコンショップのアルバイト
高校卒業前に始めたパソコンショップのアルバイトはこの時も続けていたので、大学中退後も仕事には困らなかった。
バイト先の店で社員になるという選択肢もあったが、ここでIT業界を明確に目指すことになる出来事があった。

「Webデザイナー」をやっているお客さんから、Web制作の仕事につかえるスペックのパソコンを見繕ってほしいと依頼を受けた。
よくしゃべるお客さんで、Webデザインの仕事についていろいろ教えてくれた。

ここでふと気づいた。
「自分はパソコンを売るための商品知識はあるが、パソコンを使いこなせていない」（ゲームしかしてない）

パソコン屋さんには様々なIT職の人がくる。
「プログラマー」「3DCGデザイナー」「イラストレーター」「Webデザイナー」...

お客さんの要望を聞くたびにIT業界で自分も働いてみたいと改めて思うようになった。

# 独学で勉強を始める
いろいろお客さんとから聞いた話から興味を持った仕事はこの3つ。
* プログラマー
* 3DCGデザイナー
* Webデザイナー

これらの職業に就きたいと思ってもそのスキルはないので、3つとも独学でできるところまで勉強してみることにした。

プログラマー：まっさきにあきらめた。C言語、JAVAなどの言語の書籍を何冊かかってみたが、BASICとは難易度がまるでちがって理解不能だった。またプログラマーの仕事は大変であると店のお客さんから聞き及んでいたので、ついていけると思えなかった。

3DCGデザイナー：一番楽しかったのはこれ。SHADE3Dや購入できたいくつかのツールでCGを作ってみた。楽しすぎるがゆえにこれは仕事にするよりも趣味でやった方がいいと思った。少ないバイト代で買ったPCのスペックではCGをレンダリングするには性能が足りないし、業務で使う3DMAXやMayaなどのツールは高額だったため手が出せなかった。

Webデザイナー：これがいちばんしっくり来た。HTMLは独学でもそこそこ理解できたし、Webページが少しずつ出来ていく過程がとても楽しめた。これを仕事にする貯金したバイト代でプログラミングスクール的なところに通って本格的に勉強をすることにした。

# X68000との出会い
著者が中学生になったころ、実家には父親が趣味で買ったSHAPRのX68000というパソコンがあった。これでPCゲームをして遊んでいた。

我が家はファミコンなどのゲーム機を買ってもらえない家庭だったので、ゲームで遊ぶにはパソコンを使うしかなかったのです。友達同士でのゲームの話題にはついていけなかった。ドラクエの魔法はメラくらいしか知らない。

X68000が自宅にあったことでタッチタイピングができるようになったため、のちにIT業界を目指す理由にはなったかもしれない。しかしプログラマーになるつもりはこの時点でまったく考えていなかった。あくまでもゲーム機の代替です。

## 高校生活
平凡なレベルの県立高校に入学。いわゆる「管理教育」と呼ばれる悪しき教育方針の名残のある居心地の悪い学校だった。校風にもクラスメイトにもあまりなじめず友達は少なかったが、ひとりでいる方が気楽だったので、そのように振舞っていた。

当時パソコンというものが珍しかった時代だったため、パソコンの操作ができるというだけでクラスの中では「オタク」扱いです。自分自身は別に気にしてなかったけど、どのクラスに1人はいる変な奴だったと思う。

## 勉強について
学部は普通科だったが、数学や理科が壊滅的に苦手だったため文系コースを選択。この時代にIT関連の仕事に就くには理系であることが当たり前との認識だったので、将来プログラマーになるなど1ミリも考えていなかった。

三角関数の数学のテストで3点を取り、全教科の総合得点でもクラスでドベをとってしまったことがある。クラスには1人は必ずいる「テスト勉強しなくてもこいつに負けることはないヤツ」にすら劣る成績であったことを自覚し、次回のテストからは赤点取らない程度には成績を上げました。

この程度には数学は苦手でしたので、あろうことかIT業界へ進むことなどありえないって感じでした。

## 部活の話
部活への加入が必須だっため仕方なくパソコン部へ入部。部室にはNECのPC98だったかPC88シリーズのパソコンがあり、N88 BASICでゲームらしきものを作っていた。

自宅でも部活でもパソコンを触れる環境にあったため、自然とパソコンやその業界への興味は沸き始めたのもこのころだったと思う。
BASICはそれなりにかけるようになったがC言語はポインタが理解できなかったというあるあるな理由で挫折した。

## 進路指導
高校3年の時の進路についての3者面談で「IT系の専門学校に行きたい」という希望を伝えた。パソコンは自宅でも部活でもやっていて強い関心はあったため自然な考えである。

それに対する先生と親の解答はこれだ。
「大学に行けるだけの成績があるのだから大学に行け。専門学校が大学に行けないやつが行くところだ。」
学校自体の方針として「管理教育によってバカでもそれなりにいい大学にできるように指導する」というのがあったので、当然の解答である。

僕にとってはこんなものくそくらえという気持ちだった。しかし当時の自分にはこの状況を覆すだけの自信も説得力もなく従わざるを得なかった。まだ業界そのものの知識も少なく、プログラミングのスキルだってお遊びのレベルだ。当時の自分には先生や親を説得できる材料がなかった。

ささやかな反抗として、大学受験真っただ中にこっそりパソコン屋さんでアルバイトを始めた。これがのちにIT業界を明確に目指すきっかけとなる。
こんな感じでは受験勉強に身が入るわけもなく、だれでも合格できるFラン大学に進学することになった。

# 大学でのエピソード
そんな感じでなんとなく進学したのが県内のFラン大学の商業学部は2年で中退することになります。

2年になったころ、就職活動に関する指導も学校から始まった。
しかし時は就職難真っただ中で、新卒での就職率は歴代最低水準。
先輩方は何十社も履歴書を送ってもほとんど面接にすら進めない状況。
商業学部ということもあり就職先はほぼすべて営業職。自分の望む職業ではない。
この大学を卒業したところで、就職できるかどうかすらわからず、就職できたところで望んだ職業には就けないことが分かり、これが決め手で中退をすることになった。

望む勉強もできず仕事にも就けない大学に通う目的を完全に失いました。そもそも望んでいなかった大学進学だったし。

文系学部を選んだにもかかわらず商業学部だったため「数学」「統計学」の必須科目があったことも一因でもある。


## パソコン屋さんでのアルバイト
当時はWindows95、98ブームがまだ続いていた時期で、一般家庭にパソコンとインターネット環境が普及し始めていたころです。マニア向けにはDOS/Vパーツ（PC自作パーツ）が爆売れしていたころだ。

IT系の専門学校へ進学できなかったことへの反抗として始めたパソコン屋さんのアルバイト。学校での授業よりもこの仕事の方がよほど知りたいことが学べる環境だったため、この仕事に没頭していた。

お店も大変に繁盛しており、大学を中退後もこの仕事があったため生活には困らなかったが、そろそろ就職を考えねばならない時期でした。

パソコンを買いに来たお客さんには「プログラマー」「3DCGデザイナー」「Webデザイナー」「CADオペレーター」などさまざまな職業の人たちがいた。当時の自分にはキラキラ輝いて見えた。


## 手に職を身につけよう
このまま社員として就職するという選択肢もあったが、パソコンを売る仕事ではなく「パソコンを使う仕事」をしたいと考えるようになっていた。

お客さんとの話を聞いていて、ふと気づいたことがある。
自分は「パソコン売るための商品知識はあるが、パソコンを使いこなせていない」

これまでにお客さんとから聞いた話から興味を持った仕事はこの3つ。
* プログラマー
* 3DCGデザイナー
* Webデザイナー

これらの職業に就きたいと思ってもそのスキルはないので、3つとも独学でできるところまで勉強してみることにした。

### プログラマー
真っ先にあきらめた。
BASICは高校の部活でやったことがあったが当時はJAVAやC言語というプログラミング言語が主流だった。
バイト代でいくつかの本を買って学習してみたが、理解不能だった。ポインターってなんだよって感じ。

### 3DCGデザイナー
一番楽しかったのはこれ。Shade3Dとかその手の3Dグラフィックのツールを使ってモデリングしたりして楽しんでいた。しかし、これは職業にするより趣味にした方が良いと思ったし、そもそも自分が住んでいる地域にそんな求人はなかった。

### Webデザイナー
求人もそこそこあったし、HTMLは独学でもある程度理解できたし、一番現実的な選択だと思った。
