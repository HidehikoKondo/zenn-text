# 高校生活
平凡なレベルの県立高校に入学。いわゆる「管理教育」と呼ばれる悪しき教育方針の名残のあるなんとも居心地の悪い学校だった。
数学が特に苦手で赤点常連。文系コースへ進む。英語と歴史の授業は好きだった。

部活への入部が必須だったためコンピュータ部を選択。くじ引きで部長にされる。部室にあったNECのPC88で、BASICでゲームらしきものを作った。ここではじめてプログラミングを体験した。
ここで何となくIT業界で将来仕事をするかもしれないとうっすら感じ始めたかもしれない。

自宅にはSHARP x68000というパソコンがあった。ファミコンを買ってもらえない家庭だったため、ゲーム機の代替として楽しんだ。友達とはファミコンの話題についていけず、ドラクエの魔法はメラしかわからなかった。

部活や自宅にあるパソコンがあったおかげでパソコン少年になっていた。どのクラスにも1人はいる変なヤツの爆誕です。

# 進路指導
**「IT系の専門学校にコンピュータの勉強をした」** というのが自分の希望した進路。大学受験するよりも自分の興味のある専門分野で技術を身につけたい。その方が将来の就職にも有利だと考えたからだ。

それに対して先生は強く言ってきた。
**「大学に行けるだけの成績があるのだから大学に行け。専門学校が大学に行けないやつが行くところだ。そもそもお前文系だろ」**

当時はIT業界では理系の人が就職するものという世間的な理解だったため、自分の希望進路と得意分野とはちぐはぐになっていた。
コンピュータ部に在籍していたこと程度では先生の説得材料にはならず、反論できるほどの強い主張もできず、しかたなく従うことにした。

ここで、ささやかな反抗としてパソコンショップでアルバイトをこっそり始める。
そもそも進学する気のない大学への受験勉強に身が入るわけもなく、卒業後だれでも合格できるFランク大学の商業学部へ進学することになる。

# 大学生活
なんとなく通い続けた大学の2年生になったころ、就職活動に関する指導も学校から始まった。

しかし時は就職難真っただ中で、新卒での就職率は歴代最低水準。
先輩方は何十社も履歴書を送っても面接にすら進めない状況。
商業学部ということもあり就職先はほぼすべて営業職。自分の望む職業ではない。
この大学を卒業したところで、就職できるかどうかすらわからず、就職できたところで望んだ職業には就けないことが分かり、これが決め手で中退をすることになった。

商業学部は意外と数字を扱う授業も多く、「数学」「統計学」が必須科目だった。理系科目が苦手で文系学科を選んだはずなのに、これらの単位が取れるとも思えなかったのも中退を決めた要因の一つ。

# パソコンショップのアルバイト
高校卒業前に始めたパソコンショップのアルバイトは大学中退後にも続けていたので、仕事にはあまり困らなかった。
バイト先の店で社員になるという選択肢もあったが、ここでIT業界を明確に目指すことになる出来事があった。

Webデザイナーをやっているお客さんから、Web制作の仕事につかえるスペックのパソコンを見繕ってほしいと依頼された。
よくしゃべるお客さんで、Webデザインの仕事についていろいろ教えてくれた。

ここでふと気づいた。
「自分はパソコンを売るための商品知識はあるが、パソコンを使いこなせていない」（ゲームしかしてない）
「お客さんの仕事を理解しないと本当に必要なパソコンがどれかわからない」（無駄に高いパソコンを勧めることになる）

パソコン屋さんには様々なIT職の人がくる。
「プログラマー」「3DCGデザイナー」「イラストレーター」「Webデザイナー」...

適切な商品をお勧めするためにお客さんの要望を聞いたり仕事の話を聞くたびに、パソコンショップの店員ではなく、IT業界で自分も働いてみたいと改めて思うようになっていった。

# 独学で勉強を始める
いろいろお客さんとから聞いた話から興味を持った仕事はこの3つ。
* プログラマー
* 3DCGデザイナー
* Webデザイナー

これらの職業に就きたいと思ってもそのスキルはないので、3つとも独学でできるところまで勉強してみることにした。

**プログラマー：** 真っ先に投げ出した。C言語、JAVAなどの言語の書籍を何冊かかってみたが、BASICとは難易度がまるでちがって理解不能だった。またプログラマーの仕事は大変であると店のお客さんから聞き及んでいたので、ついていけると思えなかった。

**3DCGデザイナー：** 一番楽しかったのはこれ。SHADE3Dや購入できたいくつかのツールでCGを作ってみた。楽しすぎるがゆえにこれは仕事にするよりも趣味でやった方がいいと思った。少ないバイト代で買ったPCのスペックではCGをレンダリングするには性能が足りないし、業務で使う3DMAXやMayaなどのツールは高額だったため手が出せなかった。

**Webデザイナー：** これがいちばんしっくり来た。HTMLは独学でもそこそこ理解できたし、Webページが少しずつ出来ていく過程がとても楽しめた。これを仕事にする貯金したバイト代でプログラミングスクール的なところに通って本格的に勉強をすることにした。

というわけで、アルバイトをしながらWebデザインの勉強を本格的に勉強を始めて、Webデザイナーになることを目指すのだった。
